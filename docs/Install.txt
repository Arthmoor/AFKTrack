AFKTrack Bug Tracker
--------------------

Server Requirements
-------------------

PHP 7.3.0 or greater.
MySQL 5.7 or greater.

Quick Installation Instructions
-------------------------------

If you just want to dive right in and play around, follow these steps:

* Make sure you have a database available with appropriate permissions.
* Upload the files to your web host.
* Set the following directories to 0777 permissions:
   files
   files/attachments
   files/emojis
* Set permissions to 0777 on all of the default emoji files as well.
   files/posticons
   files/banners
* Set the default logo to 0777 permissions:
   files/banners/afktracklogo.png

* Point your browser to https://yoururl/whereafktrackis/install

Follow the instructions in the installer. Remove the install folder when the install is complete.
